## What is an Expression?

An **expression** is a piece of code that produces a value.  
It can be made up of:
- **Literals** (like `5` or `"hello"`)
- **Variables** (like `x` or `name`)
- **Operators** (like `+`, `-`, `*`, `/`)
- **Function calls** (like `len("hi")`)

##  Examples

- `2 + 3` â†’ evaluates to `5`
- `"hi" + " there"` â†’ evaluates to `"hi there"`
- `x * 4` â†’ if `x = 2`, this evaluates to `8`
- `len("cat")` â†’ evaluates to `3`

# Expression vs Statement

## Expression
- An **expression** is code that produces a **value**.  
- It can be used inside other code.  
- Examples:
  - `2 + 3` â†’ gives `5`
  - `"hi".upper()` â†’ gives `"HI"`
  - `x * 4` â†’ gives some number, depending on `x`

## Statement
- A **statement** is code that **does something**.  
- It usually controls the flow of the program or makes Python take an action.  
- Examples:
  - `x = 5` â†’ assignment statement (stores a value in `x`)
  - `if x > 0:` â†’ if statement (controls code execution)
  - `for i in range(3):` â†’ loop statement
